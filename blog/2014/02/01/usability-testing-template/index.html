ユーザビリティテストのタスクシート

2014/02/01

ユーザビリティテスト中に使用するメモ用紙の項目をメモするためのテンプレート（雛形）を記載しています。ただユーザビリティテストを専門で行っているわけではないので、要素の抜け漏れなどはあるかもしれません。ここではユーザビリティ全体ではなく、テスト時に使用するタスクシートに関して記載しています。ユーザビリティのプラン作成やレポート作成、リクルーティングなどは参考情報をご参照下さい。


1.タスクシートの雛形

https://docs.google.com/spreadsheet/ccc?key=0ApZE_qOGDfNHdEYySkRNOGRBR0U0N2s3T2U0azJwelE&usp=sharing

2.項目

達成

  • ◯はタスクを完了した場合
  • △はインタビューアーのヒントを得てタスクを完了した場合
  • ☓はタスクを完了で来なかった場合

効率

  • ◯は想定の操作通りにできた場合
  • △は多少迷っていた場合
  • ☓は目安時間を超過した場合

満足

  • ◯は被験者が特に不満を持っていないと思われた場合
  • △は不満を持っていそうだと思われた場合
  • ☓は不満を露わにしていた場合

評価

達成・効率・満足を数値化してどの操作（タスク）が問題となっていたかを把握します。設定してる係数は以下の表の通りです。

　　 達成 効率 満足
◯ 0 0 0
△ 3 2 1
☓ 5 4 3
